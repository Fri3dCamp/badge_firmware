# Fri3d badges firmware

This repository holds the main firmware for the Fri3d Camp badges.

## Building

### Requirements

* [esp-idf](https://github.com/espressif/esp-idf) v5.2.2

### Fox badge (2024)

```shell
cd boards/fox
idf.py flash
```

### Octopus badge (2022)

```shell
cd boards/octopus
idf.py flash
```
